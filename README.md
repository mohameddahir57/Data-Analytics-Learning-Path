##  Full Course Outline

###  Module 1 Introduction to Data Analytics
**Duration: 1 Week**

| Week | Class Day 1 | Class Day 2 | Assignment |
|------|------------|------------|------------|
| Week 1 | Meaning of data analytics, role of a data analyst, types of analytics, analytics workflow | Data sources, structured vs unstructured data, basic data ethics, spreadsheets vs databases vs BI tools | Write a short reflection on the analytics lifecycle and classify 10 example datasets by data type and source |

###  Module 2 Microsoft Excel for Data Analytics
**Duration: 3 Weeks (Weeks 2–4)**

| Week | Class Day 1 | Class Day 2 | Assignment |
|------|------------|------------|------------|
| Week 2 | Excel interface, tables, data types, sorting, filtering, freeze panes, workbook organization | Data cleaning with text functions, flash fill, remove duplicates, handling blanks, data validation | Clean a messy sales dataset and submit a before/after worksheet with explanation of steps used |
| Week 3 | Formulas: SUM, AVERAGE, MAX, MIN, COUNT, IF, COUNTIF, SUMIF, lookup functions (VLOOKUP, XLOOKUP), date functions | PivotTables, PivotCharts, slicers, grouping, and summary analysis | Apply formulas to a sales dataset and build a PivotTable with chart and slicers |
| Week 4 | Dashboard design in Excel, charts, KPI cards, interactive filters, layout best practices | Introduction to Power Query for importing, transforming, appending, and reshaping data | Build a one-page Excel dashboard with at least 2 charts, 3 KPI cards, and a slicer |

###  Module 3 Power BI for Data Analytics
**Duration: 2 Weeks (Weeks 5–6)**

| Week | Class Day 1 | Class Day 2 | Assignment |
|------|------------|------------|------------|
| Week 5 | Power BI interface, importing data, Power Query basics, data transformation workflow | Relationships, data modeling, star schema basics, measures vs calculated columns, first visuals | Import a dataset, transform it, build a clean model, and create one report page with core KPIs |
| Week 6 | DAX Basics: SUM, COUNT rows, CALCULATE, DIVIDE, Time-Aware Thinking, KPI Measures | Dashboard design, drill-through, filters, publishing workflow, and presentation of insights | Develop a 2-page Power BI dashboard with 5 measures and a short business insight summary |

###  Module 4 Tableau for Data Visualization
**Duration: 2 Weeks (Weeks 7–8)**

| Week | Class Day 1 | Class Day 2 | Assignment |
|------|------------|------------|------------|
| Week 7 | Connecting data in Tableau, dimensions vs measures, worksheets, basic chart building, formatting | Sorting, filtering, groups, sets, calculated fields, effective chart selection | Create 4 worksheets from one business dataset and explain the purpose of each visualization |
| Week 8 | Dashboards, containers, actions, parameter basics, and storytelling principles | Interactive dashboard design, tooltips, dashboard layout, and presentation of findings | Build a Tableau dashboard with at least 3 worksheets, one filter action, and one story point |

###  Module 5 SQL for Data Analytics
**Duration: 4 Weeks (Weeks 9–12)**

| Week | Class Day 1 | Class Day 2 | Assignment |
|------|------------|------------|------------|
| Week 9 | Introduction to databases, SQL environment, SELECT statements, filtering with WHERE, ORDER BY, LIMIT | Aliases, arithmetic calculations, logical conditions, IN, BETWEEN, LIKE, NULL handling | Write 12 SQL queries to explore a sample customer-orders database and submit results screenshots |
| Week 10 | Aggregate functions: COUNT, SUM, AVG, MIN, MAX, and GROUP BY | HAVING clause, grouping best practices, business summary reporting | Prepare a grouped summary report answering revenue, quantity, and customer performance questions |
| Week 11 | Inner Join, Left Join, right concept explanation, joining multiple tables | Entity relationships, data quality issues in joins, combining fact and dimension tables | Join at least 3 tables and produce a clean analytical output for a sales performance case |
| Week 12 | Subqueries, CASE WHEN, CTE introduction, reusable query structure | Window function concepts for ranking, running totals, and final review for analyst reporting | Build a mini SQL project including joins, aggregation, CASE logic, and at least one advanced query |

###  Module 6 RStudio for Data Analytics
**Duration: 4 Weeks (Weeks 13–16)**

| Week | Class Day 1 | Class Day 2 | Assignment |
|------|------------|------------|------------|
| Week 13 | R and RStudio environment, scripts, packages, data frames, vectors, importing CSV/Excel files | Data cleaning basics in R using dplyr: select, filter, mutate, rename, arrange | Import a dataset in RStudio and produce a cleaned version with a short script explanation |
| Week 14 | Grouping and summarizing data with dplyr, creating derived variables, tidy data concepts | Data visualization with ggplot2: bar chart, line chart, histogram, boxplot, visual interpretation | Generate 4 charts in R and write a one-page interpretation of the key patterns |
| Week 15 | Descriptive statistics, correlation, t-test, and introduction to simple linear regression in R | Running and interpreting basic statistical output, checking assumptions, communicating results | Analyze a small dataset in R and submit descriptive statistics plus one simple statistical test |
| Week 16 | Reproducible analysis workflow, R Markdown/Quarto concept, combining code, output, and explanation | Final capstone guidance: choose tool, clean data, analyze, visualize, present recommendations | **Final Capstone:** Submit a mini end-to-end analytics project using Excel, Power BI, SQL, Tableau, or RStudio |
